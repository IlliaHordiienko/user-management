## <ins>Description</ins>
This project is a <b>User Management System</b> developed as a university practical assignment. It demonstrates core Object-Oriented Programming (OOP) concepts in PHP.

## <ins>Core Features</ins>
* Secure login/logout system using <b>PHP Sessions</b>.
* Safe password handling using `password_hash()`.
